Module Bundlers

Los module bundlers o empaquetadores de módulos JavaScript se encargan de agrupar varios módulos en uno o más paquetes optimizados para el navegador permitiendo a los desarrolladores escribir código modular con los beneficios que genera.

Además, también se encargan de realizar tareas que compilen, concatenen y minifiquen hojas de estilos y scripts, junto con optimización de imágenes y assets para el entorno de producción.

Principales Module Bundlers del mainstream actual

Parcel

Parcel es un empaquetador de aplicaciones web que se diferencia del resto por la experiencia de uso a los desarrolladores, ...

Webpack

Es un empaquetador de módulos que permite la gestión de dependencias, conversión de formatos, servidor de desarrollo, etc.
